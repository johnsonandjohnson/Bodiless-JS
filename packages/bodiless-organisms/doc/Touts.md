# Touts

- Purpose
  - Provides a group on components commonly used as blocks on a site.
- Properties
  - none
- Editable areas
  - Title, Body, Image, CTA

## Usage

```js
import { Tout, ImgTout, HTout } from '@bodiless/organisms';

<Tout nodeKey="tout1" />
<ImgTout nodeKey="tout6" />
<HTout nodeKey="tout4" />
```
