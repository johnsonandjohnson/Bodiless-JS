Composable Token which makes the non-GatsbyImage component full-width.

When adding a new image into a page, it is placed as a normal tag instead of a GatsbyImage component. After refreshing the page, it is rendered as a GatsbyImage, which is full-width by default. Although not desirable, this behavior is expected.

Applying this token makes this tag full-width, so it renders just like a GatsbyImage. This is only required if you don't want the image to "change its size" when refreshing the page after placing it.
